Dynamic Inventory

AWS EC2 inventory plugin — auto-discover instances by tag and region.

Dynamic inventory generates host lists from cloud APIs — eliminating manual IP edits when instances churn.

Core Model

Inventory Plugins

Plugin

ansible.builtin.aws_ec2 — query EC2 API.

constructed

Build groups from instance tags.

cache

Inventory cache reduces API calls.

aws_profile

Credential chain like AWS CLI.

hostnames

Public vs private DNS selection.

ansible-inventory

Validate plugin output offline.

EC2 Plugin

aws_ec2 Flow

Repo lab files: ansible/extended/labs/dynamic_inventory/
aws_ec2.yml
plugin: amazon.aws.aws_ec2
regions:
  - us-east-1
filters:
  tag:Environment: dev
keyed_groups:
  - key: tags.Role
    prefix: role
Run
ansible-inventory -i aws_ec2.yml --graph
