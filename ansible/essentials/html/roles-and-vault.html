Roles & Vault

Role directory layout, ansible-galaxy, and encrypting secrets with Ansible Vault.

Roles package tasks, handlers, templates, and defaults. Ansible Vault encrypts sensitive files at rest in Git.

Core Model

Roles and Secrets

Role layout

tasks/, handlers/, templates/, defaults/, vars/.

roles_path

ansible.cfg roles_path or ./roles/

include_role

Dynamic role inclusion in tasks.

Vault

Encrypt files — ansible-vault encrypt secrets.yml.

Vault password

File, prompt, or script — never commit password.

Lab 07 combines roles, vault, and Node.js deploy.

Vault

Vault Operations

Create
ansible-vault create secrets.yml
Edit
ansible-vault edit secrets.yml
Encrypt string
ansible-vault encrypt_string 'dbpass' --name 'db_password'
Vault ID file
echo 'mypass' > ~/.vault_pass && chmod 600 ~/.vault_pass
