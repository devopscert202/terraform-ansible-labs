Playbook Execution & Handlers

Tasks run in order; handlers run once at the end if notified

1
Install apache2 (apt module)
Reports changed if package was missing
2
Enable mod_rewrite + notify: Restart apache2
Queues handler; does not restart immediately
3
End of play
All tasks finished on all hosts
H
Handler: Restart apache2
Runs only if step 2 changed something
- name: Enable mod_rewrite
  ansible.builtin.apache2_module:
    name: rewrite
    state: present
  notify: Restart apache2

handlers:
  - name: Restart apache2
    ansible.builtin.service:
      name: apache2
      state: restarted
