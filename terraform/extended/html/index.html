Safe start — run from any lab directory after authenticating: 
export AWS_PROFILE=your-training-profile
aws sts get-caller-identity
terraform version
terraform init && terraform validate
Credentials belong in the AWS credential chain or IAM role — never in .tf files.

❓ FAQ

QuestionAnswer
Do I need AWS for every lab?No. Labs 02, 04, 06, and 07–11 support validate-only paths; 14–15 need AWS for apply.
Where do credentials go?AWS_PROFILE, environment variables, or IAM role — never in committed .tf files.
Can I skip state labs?Not recommended. Labs 07–11 teach team-critical operations.
What about Terraform Cloud?These labs use CLI + S3; concepts transfer to TFC remote backends.
How do I open HTML guides?Open terraform/extended/html/index.html in a browser from the repo clone.
