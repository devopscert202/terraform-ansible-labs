# Terraform Extended

A hands-on continuation of Terraform Essentials covering providers, provisioners, workspaces, shared state, collections, functions, dynamic blocks, and capstone projects. Terraform 1.5+ is required. AWS examples use the normal AWS credential chain (`AWS_PROFILE`, environment credentials, or an attached IAM role); no AWS access keys are stored in source.

Start with [lab manuals](labmanuals/README.md), then run the matching directory in `labs/`. Destroy cloud resources when you finish.
