Local State

What state contains, why it matters, and local state operations in Lab 06.

State is Terraform's memory. It maps aws_instance.web in HCL to i-0abc123 in AWS — without it, Terraform cannot plan updates safely.

Core Model

Understanding State

Resource mapping

Links configuration addresses to real cloud resource IDs.

Metadata

Stores dependency info, serial numbers, and sometimes sensitive attributes.

Local default

terraform.tfstate lives in the working directory — fine for solo labs.

Never edit casually

Manual state surgery is for experts; prefer import and refresh.

Gitignore

State often contains secrets — add terraform.tfstate* to .gitignore.

State commands

state list, state show, state mv for operations.

Hygiene

State Safety Practices

.gitignore
terraform.tfstate
terraform.tfstate.backup
.terraform/
*.tfvars
Backup before risky ops
cp terraform.tfstate terraform.tfstate.backup.manual
Sensitive output
output "db_password" {
  value     = random_password.db.result
  sensitive = true
}
