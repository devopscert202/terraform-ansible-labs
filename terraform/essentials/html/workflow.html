The loop

terraform init
terraform fmt -recursive
terraform validate
terraform plan -out=tfplan
terraform apply tfplan
terraform destroy

validate checks configuration structure. A successful validation does not guarantee AWS permissions, quota, or that an apply will succeed. The plan is the change-review artifact.
